.formData()

Extracts data from the first form in the collection.

Examples

1. Get native FormData object

const formData = $('#myForm').formData();

Result

// Returns native FormData instance
FormData { ... }

2. Get plain object (Serialize)

const data = $('#myForm').formData(true);

Result

{
     "username": "john",
     "password": "123",
     "roles": ["admin", "editor"]
}

Parameters

Parameter Type Description
serialize boolean If true, returns a plain JavaScript object (similar to serializeArray). If false (default), returns a native FormData object.

Returns: FormData or Object.
