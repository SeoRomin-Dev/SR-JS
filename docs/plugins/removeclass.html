.removeClass()

Remove a single class, multiple classes, or all classes from each element in the set of matched elements.

Examples

1. Remove one class

// <div class="box active"></div>
$('div').removeClass('active');

Result

<div class="box"></div>

2. Remove multiple

// <div class="box active visible"></div>
$('div').removeClass('active visible');

Result

<div class="box"></div>

3. Remove all

// <div class="box active"></div>
$('div').removeClass();

Result

<div class=""></div>

Parameters

Parameter Type Description
className string Optional. One or more space-separated classes to remove. If omitted, all classes are removed.

Returns: The original SR object for chaining.
